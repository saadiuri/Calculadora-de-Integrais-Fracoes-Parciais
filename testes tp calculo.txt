Caso 1: Fatores Lineares Distintos

  - Teste 1.1 (Simples)

      Numerador: 1

      Denominador: (x-1)(x-2)

      Integral Original: ∫ (1) / ((x-1)(x-2)) dx

      Análise do Denominador: CASO 1: Fatores Lineares Distintos (r1=1, r2=2)

      Decomposição em Frações Parciais: -1/(x - 1) + 1/(x - 2)

      Integração Termo a Termo: ∫ (-1/(x - 1)) dx + ∫ (1/(x - 2)) dx

      Resultado Simbólico Final: -1*ln|x - 1| + 1*ln|x - 2| + C

  - Teste 1.2 (Numerador com 'x')

      Numerador: 5x - 5

      Denominador: (x-3)(x+2)

      Integral Original: ∫ (5x - 5) / ((x-3)(x+2)) dx

      Análise do Denominador: CASO 1: Fatores Lineares Distintos (r1=3, r2=-2)

      Decomposição em Frações Parciais: 2/(x - 3) + 3/(x + 2)

      Integração Termo a Termo: ∫ (2/(x - 3)) dx + ∫ (3/(x + 2)) dx

      Resultado Simbólico Final: 2*ln|x - 3| + 3*ln|x + 2| + C

  - Teste 1.3 (Numerador com 'x' e números maiores)

      Numerador: 4x + 16

      Denominador: (x+1)(x-5)

      Integral Original: ∫ (4x + 16) / ((x+1)(x-5)) dx

      Análise do Denominador: CASO 1: Fatores Lineares Distintos (r1=-1, r2=5)

      Decomposição em Frações Parciais: -2/(x + 1) + 6/(x - 5)

      Integração Termo a Termo: ∫ (-2/(x + 1)) dx + ∫ (6/(x - 5)) dx

      Resultado Simbólico Final: -2*ln|x + 1| + 6*ln|x - 5| + C

      Integração Termo a Termo: ∫ ((0.3333x)/(x²+1)) dx + ∫ ((-0.3333x)/(x²+4)) dx

      Resultado Simbólico Final: 0.1667 * ln|1x² + 0x + 1| + -0.1667 * ln|1x² + 0x + 4| + C

Caso 4: Quadrático Repetido

  - Teste 4.1 (Numerador 'x')

      Numerador: x

      Denominador: (x^2+1)^2

      Integral Original: ∫ (x) / (x^2+1)^2 dx

      Análise do Denominador: CASO 4: Fator Quadrático Repetido (x²+1)²

      Decomposição em Frações Parciais: (0x + 0)/(x² + 1) + (1x + 0)/(x² + 1)²

      Integração Termo a Termo: ∫ (0/(x²+1)) dx + ∫ (x/(x²+1)²) dx

      Resultado Simbólico Final: (-0.5000) / (x² + 1) + C

  - Teste 4.2 (Numerador constante)

      Numerador: 1

      Denominador: (x^2+1)^2

      Integral Original: ∫ (1) / (x^2+1)^2 dx

      Análise do Denominador: CASO 4: Fator Quadrático Repetido (x²+1)²

      Decomposição em Frações Parciais: (0x + 0)/(x² + 1) + (0x + 1)/(x² + 1)²

      Integração Termo a Termo: ∫ (0/(x²+1)) dx + ∫ (1/(x²+1)²) dx

      Resultado Simbólico Final: 0 * (x / (x² + 1)) + 0.5000 * arctan(x/1) + 0.5000 * (x / (x² + 1)) + C (Simplificado: 0.5*arctan(x) + 0.5*x/(x²+1) + C)

  - Teste 4.3 (Numerador 'x^3 + ...')

      Numerador: x^3 + 1x^2 + 3x + 1

      Denominador: (x^2+1)^2

      Integral Original: ∫ (x^3 + 1x^2 + 3x + 1) / (x^2+1)^2 dx

      Análise do Denominador: CASO 4: Fator Quadrático Repetido (x²+1)²

      Decomposição em Frações Parciais: (1x + 1)/(x² + 1) + (2x + 0)/(x² + 1)²

      Integração Termo a Termo: ∫ ((1x + 1)/(x² + 1)) dx + ∫ ((2x + 0)/(x² + 1)²) dx

      Resultado Simbólico Final: 0.5000*ln|x²+1| + 1*arctan(x/1) + (-1) / (x² + 1) + C
